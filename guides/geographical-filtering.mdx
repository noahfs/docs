---
title: "Geographical Filtering"
---

Certain Glimpse API endpoints support filtering the results by a `geo` parameter, consisting of a two-letter country code. Below, we offer a list of all available codes for the Glimpse APIs.

[ Download glimpse\_enterprise\_api\_geos.csv](https://drive.google.com/file/d/12S2j9CEeHyXFA8gohcrArSk8BBP8_soi/view?usp=sharing)

To search by a country, use the country code, like this: `US`

When no geo parameter is specified, worldwide data is returned.
